alpine-redis
===============
A Redis image with a small disk footprint (~7MB) for Redis 3.0.5

|Layers| |Stars| |Pulls|

Example Usage
-------------

.. code-block::

    docker run --name redis -d -p 6379:6379 registry.awebertest.com/aweber/alpine-redis
